# 东方魔炮使 (Touhou Blaster Master)

> 工作标题：东方魔炮使 / Touhou Blaster Master
> 版本：0.1
> 最后更新：2026-03-26
> 支持语言：简体中文 (zh_CN), 日本語 (ja_JP), English (en_US)

---

## 1. 概述 (Overview)

一款东方 Project 粉丝向弹幕射击增量游戏。玩家操控雾雨魔理沙，在 10-30 秒的短战斗循环中发射魔炮光束击败敌人、收集资源，然后在进化树场景中消耗资源点亮节点变强。从"细弱手动魔炮"成长到"满屏自动光束"，体验爆炸式的正反馈爽快感。

**核心特色**：
- 单局 10-30 秒的超短循环，随时可停
- 网状进化树，自由构建 Build
- 擦弹加速蓄力 + 被弹中断的风险/回报机制
- 东方同人风格轻量剧情（看似大冲突，实则小矛盾/无厘头）
- 像素 2D 美术，目标平台 PC + Web
- 多语言支持：简体中文、日本語、English

---

## 2. 玩家幻想 (Player Fantasy)

- **成为魔炮大师**：在密集弹幕中穿梭，擦弹如舞蹈，一发光束清屏
- **成长可视化**：从弱小的细光束 → 多重穿透光束 → 满屏自动扫射
- **东方沉浸**：幻想乡架空场景、原作角色互动、符卡收集
- **轻松体验**：无失败惩罚，只有收集效率差异，适合碎片时间

## 3. 详细规则 (Detailed Rules)

### 3.1 游戏循环

```
┌─────────────────────────────────────────────────────────────┐
│                     游戏循环                                 │
│                                                             │
│   ┌──────────────┐         ┌──────────────┐                │
│   │  战斗场景    │ ──────→ │  进化树场景   │                │
│   │  (10-30 秒)   │  收集   │  (玩家主动)   │                │
│   │  魔理沙魔炮  │  资源   │  消耗资源点亮  │                │
│   └──────↑───────┘         └──────┬───────┘                │
│         │                         │                        │
│         └─────────────────────────┘                        │
│                   继续战斗                                  │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 战斗场景规则

| 规则 | 说明 |
|------|------|
| **时长** | 初始 10 秒，可通过进化树升级延长，最长 30 秒 |
| **返回条件** | 时间结束自动返回 OR 玩家主动放弃返回进化树 |
| **失败条件** | 无失败。被弹仅导致蓄力中断、提前发射、光束变弱 |
| **操作方式** | 移动躲避 + 擦弹 + 按住空格蓄力 + 释放发射 |
| **资源收集** | 初期需接触掉落物，可升级收集范围，最终自动收集 |
| **敌人设计** | 多类型多模式，各有不同弹幕 pattern |
| **精英敌人** | 高血量、高弹幕密度，掉落辅助资源 |

### 3.3 进化树场景规则

| 规则 | 说明 |
|------|------|
| **结构** | 网状树，多路径选择 |
| **节点类型** | 普通节点、精英节点、敌人升级节点、阶段节点 |
| **返回战斗** | 玩家自由选择何时返回，无次数限制 |
| **重置机制** | [待定 - 是否允许重置进化树] |

### 3.4 资源系统

| 资源类型 | 来源 | 用途 |
|----------|------|------|
| **主要资源** | 所有敌人击败后掉落 | 点亮普通节点、敌人升级节点 |
| **辅助资源** | 精英敌人击败后掉落 | 解锁精英节点、阶段节点 |
| **特殊货币** | 成就/里程碑奖励 | 解锁新角色、皮肤等（可选） |

### 3.5 魔炮光束系统

| 属性 | 升级路径 |
|------|----------|
| **粗细** | 细 → 中 → 粗 → 超粗（覆盖范围递增） |
| **伤害** | 低 → 中 → 高 → 秒杀杂兵 |
| **持续时间** | 短 → 中 → 长 → 持续光束 |
| **数量** | 单光束 → 双光束 → 三重 → 散射 |
| **特效** | 无 → 穿透 → 爆炸 → 追踪 → 连锁 |
| **控制** | 手动蓄力 → 可切换自动 → 完全自动 |

### 3.6 擦弹与被弹机制

| 行为 | 效果 |
|------|------|
| **擦弹成功** | 擦过弹幕边缘不受伤，蓄力速度加快 |
| **被弹** | 蓄力中断，光束提前发射，威力减半（粗细/时长/伤害降低） |
| **无敌时间** | [待定 - 被弹后是否有短暂无敌] |

## 4. 公式 (Formulas)

> [待定 - 需在系统设计阶段细化]

### 4.1 蓄力公式

```
蓄力时间 = 基础蓄力时间 × (1 - 擦弹加速加成 - 升级加速加成)
```

### 4.2 光束强度公式

```
光束强度 = 基础伤害 × 粗细倍率 × 数量倍率 × 特效加成
```

### 4.3 资源掉落公式

```
资源掉落 = 基础掉落 × 敌人类型系数 × 敌人升级节点倍率
```

### 4.4 敌人血量公式

```
敌人血量 = 基础血量 × (1 + 敌人升级节点次数 × 0.2)
```

| 情况 | 处理方式 |
|------|----------|
| **战斗中被弹中断** | 立即发射弱化光束，本局仍可继续 |
| **战斗时间结束** | 自动返回进化树，资源保留 |
| **主动放弃战斗** | 立即返回进化树，已收集资源保留 |
